## 📌 Overview  

This project performs **Exploratory Data Analysis (EDA)** on Diwali sales data to uncover customer purchasing behavior and business insights.  

The goal is to help businesses understand **which customers are more likely to buy products during Diwali** and improve marketing strategies.

---

## 🎯 Objectives  

- Analyze customer demographics  
- Identify top-performing product categories  
- Understand purchasing patterns  
- Find high-value customer segments  

---

## 📂 Dataset Features  

The dataset includes:  

- User_ID  
- Gender  
- Age Group  
- Marital Status  
- State  
- Occupation  
- Product Category  
- Amount (Purchase Value)  

---

## 🛠️ Tools & Libraries Used  

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 🔍 Analysis Performed  

- Data Cleaning (handling missing values, dropping irrelevant columns)  
- Gender-wise analysis  
- Age group analysis  
- State-wise sales analysis  
- Occupation-based insights  
- Product category analysis  
- Top customers identification  

---

## 📊 Key Insights  

- 👩 Female customers spend more than male customers  
- 🎯 Age group **26-35 years** has the highest purchasing power  
- 🏙️ Top states contributing to sales: Uttar Pradesh, Maharashtra, Karnataka  
- 💼 Working professionals (IT, Healthcare, Aviation) are major buyers  
- 🛍️ Food, Clothing, and Electronics are top-selling categories  
- 💍 Married customers have higher purchasing power  

---

## 🖥️ Sample Visualizations  

- Bar charts for gender and age analysis  
- Count plots for categorical variables  
- Heatmaps for correlations  
- Sales distribution plots  

---

## 🎯 Business Insights  

- Target **female customers aged 26-35** for better sales  
- Focus marketing campaigns in **top-performing states**  
- Promote high-demand product categories  
- Personalize offers for working professionals
